<h1>url-shortener</h1>
The code snippet demonstrates a basic URL shortener API using Flask and MySQL.
<h2>Key components include:</h2>

<h2>Flask Setup:</h2> The app initializes routes for rendering templates (/, /index) and handling URL shortening (/shorten). It also configures MySQL for database storage.
<h2>Database Configuration:</h2> The MySQL settings (host, user, password, database) are specified. The urls table stores original URLs, short codes, and visitor counts.
<h2>URL Shortening Logic:</h2> When a user submits an original URL, the code generates a short code using pyshorteners (TinyURL). It inserts data into the database.
<h2>Redirection Logic:</h2> Accessing a short URL triggers a lookup in the database. If found, the user is redirected to the original URL.
